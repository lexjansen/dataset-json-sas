Detailed Description

Create a dataset with directory information.

credit: https://communities.sas.com/t5/SAS-Programming/listing-all-files-of-all-types-from-all-subdirectories/m-p/334113/highlight/true#M75419

Example usage: 

%util_gettree(
  dir=&project_folder/json_out/sdtm, 
  outds=work.dirtree_sdtm, 
  where=%str(ext="json" and dir=0)
);
Author
Lex Jansen
Parameters
[in]dir=The starting directory
[out]outds=(work.dirtree) Output dataset
[in]where=Filter records
[in]keep=Variables to keep in output dataset
[in]drop=(ext) Variables to drop in output dataset
