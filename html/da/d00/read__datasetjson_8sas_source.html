1/**
2 @file read_datasetjson.sas
3 @brief Read a Dataset-JSON file to a SAS dataset.
4
5 @details This macro creates a SAS dataset from a Dataset-JSON file<br />
6 Optionally, the dataset-JSON metadata is saved in a number of metadata tables:
7 @li metadata_study
8 @li metadata_tables
9 @li metadata_columns
10
11 Example usage:
12
13 %read_datasetjson(
14 jsonpath=&project_folder/json_out/sdtm/dm.json,
15 datalib=datasdtm,
16 savemetadata=N
17 );
18
19 %read_datasetjson(
20 jsonpath=&project_folder/json_out/sdtm/dm.json,
21 datalib=datasdtm
22 savemetadata=Y,
23 metadatalib=metasdtm);
24
25 @author Lex Jansen
26
27 @param[in] jsonpath= Path to Dataset-JSON file
28 @param[in] jsonfref= File reference for the Dataset-JSON file. Either jsonpath or jsonfref has to be sppecified.
29 @param[out] datalib= Library to save SAS data set
30 @param[in] savemetadata= (Y) Use Define-XML metadata? (Y/N)
31 @param[out] metadatalib= (work) Library to save the metadata datasets
32 The following datasets are saved:
33 @li metadata_study
34 @li metadata_tables
35 @li metadata_columns
36
37**/
