1/**
2 @file write_datasetjson.sas
3 @brief Write a SAS dataset to a Dataset-JSON file.
4
5 @details This macro creates a Dataset-JSON file from a SAS dataset<br />
6 Metadata is taken from the dataset or from a number of metadata tables:
7 @li metadata_study
8 @li metadata_tables
9 @li metadata_columns
10
11 Some metadata can be specified as parameters
12 (fileOID, originator, sourceSystem, sourceSystemVersion,
13 studyOID, metaDataVersionOID, metaDataRef, datasetlabel).
14
15 Example usage:
16
17 %write_datasetjson(
18 dataset=datasdtm.dm,
19 jsonpath=&project_folder/json_out/sdtm/dm.json);
20
21 %write_datasetjson(
22 dataset=datasdtm.dm,
23 jsonpath=&project_folder/json_out/sdtm/dm.json,
24 usemetadata=Y,
25 metadatalib=metasdtm);
26
27 @author Lex Jansen
28
29 @param[in] dataset= (libname.)memname of the SAS data set
30 @param[out] jsonpath= Path to Dataset-JSON file
31 @param[out] jsonfref= File reference for the Dataset-JSON file.
32 Either jsonpath or jsonfref has to be sppecified.
33 @param[in] usemetadata= (N) Use Define-XML metadata? (Y/N)
34 @param[in] metadatalib= Define-XML metadata library
35 The following datasets are expected:
36 @li metadata_study
37 @li metadata_tables
38 @li metadata_columns
39 @param[in] decimalVariables= List of numeric variables to write as decimal strings.
40 Not used when usemetadata=Y. Separated by blanks.
41 @param[in] datasetJSONVersion= (1.1.0) Dataset-JSON version. Allowed values: 1.1.*
42 @param[in] originator= The organization that generated the Dataset-JSON dataset.
43 @param[in] sourceSystem= The name of the information system from which the content of this dataset was sourced
44 @param[in] sourceSystemVersion= The version of the information system from which the content of this dataset was sourced
45 @param[in] studyOID= Unique identifier for the study that may also function as a foreign key to a Study/\@OID in an associated Define-XML file
46 @param[in] metaDataVersionOID= Unique identifier for the metadata version that may also function as a foreign key to a MetaDataVersion/\@OID in an associated Define-XML file
47 @param[in] metaDataRef= URI for a metadata file describing the dataset (e.g., a Define-XML file)
48 @param[in] datasetlabel= Dataset label
49 @param[in] pretty= (NOPRETTY) Format Dataset-JSON file (PRETTY/NOPRETTY).
50
51 <h4> Related Macros </h4>
52 @li write_datasetjson_1_0.sas
53 @li write_datasetjson_1_1.sas
54
55**/
